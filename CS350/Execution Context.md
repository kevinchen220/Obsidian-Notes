# Execution Context

> [!tldr] Environment where functions execute
> Includes
> * arguments
> * local variables
> * memory

**There are multiple execution contexts**
* Application context
	* Application threads
* Kernel Context
	* [[Kernel Thread|Kernel threads]], software [[Interrupts]]
* [[Interrupts|Interrupt]] context
	* Interrupt handler
